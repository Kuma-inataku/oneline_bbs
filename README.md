### ifの使い方（主にバリデーション）
elseは使わずにエラー処理時はif(!~)などで早々にエラー処理する
 →ソースコード読者が正常処理だけに意識すればいいように考慮

ex.
if ( // エラー条件1 ) {
  // エラー処理1
} elseif (エラー条件2) {
  // エラー処理2
} else {
  正常処理
}

↓
